Features & Besonderheiten

Was JarvMC besonders macht

7 einzigartige Features, die uns von anderen Minecraft-Servern unterscheiden.

Feature #1

Self-Hosted auf Arch Linux

Keine teuren Hosting-Anbieter, keine Shared-Hosting-Probleme. Wir kontrollieren jede einzelne Zeile Code auf unserem eigenen Arch-System.

  • Eigene Hardware-Kontrolle
  • Maßgeschneiderter Linux-Kernel
  • Keine "Noisy Neighbor"-Probleme
  • Volle Root-Zugriff
Feature #2

Zero-Downtime Updates

Dank unserer Container-Architektur mit Docker können wir Updates durchführen, ohne den Server auch nur eine Sekunde zu stoppen.

  • Docker-Container für nahtlose Updates
  • Rolling Updates ohne Disconnects
  • Automatische Rollbacks bei Fehlern
  • 99.9% Uptime garantiert
Feature #3

Eigene Plugins

Wir programmieren unsere eigenen Features in Java statt Standard-Plugins zu nutzen. Das bedeutet: Exakt das, was wir wollen.

  • Maßgeschneiderte Gameplay-Mechanics
  • Keine überflüssigen Features
  • Optimale Performance
  • Schnelle Bugfixes
Feature #4

Wöchentliche Events

Jeden Samstag etwas Neues! Von epischen Boss-Kämpfen über kreative Bau-Wettbewerbe bis hin zu nervenaufreibenden Parkour-Challenges.

Build-Wettbewerbe

Feature #5

100% No Pay2Win

Bei uns gibt es keine kosmetischen Items, die dir einen Vorteil verschaffen. Fairness steht an erster Stelle.

  • Keine OP-Items zum Kaufen
  • Keine Pay-to-Skip-Mechaniken
  • Keine Lootboxen
  • Reines Skill-basiertes Gameplay
Feature #6

Bluemap 3D-Ansicht

Nicht nur eine 2D-Karte – erkunde die Welt in voller 3D-Pracht direkt im Browser. Zoome durch Höhlen, fliege über Berge, entdecke verborgene Strukturen.

  • Vollständige 3D-Renderings
  • Höhlenansicht verfügbar
  • Echtzeit-Player-Tracking
  • Marker für wichtige Orte
Feature #7

Discord-Minecraft Chat

Bleib immer verbunden! Schreibe von Discord direkt mit Spielern im Spiel und umgekehrt. Vollständige Chat-Synchronisation in beide Richtungen.

  • Zwei-Wege-Chat-Sync
  • Discord → Minecraft Nachrichten
  • Minecraft → Discord Nachrichten
  • Server-Status Updates

Weitere Features

CoreProtect

Jede Block-Änderung wird protokolliert. Griefing kann vollständig rückgängig gemacht werden.

Plot-System

Sichere Grundstücke mit individuellen Berechtigungen für Freunde.

Spieler-Shops

Erstelle deinen eigenen Shop und verkaufe Items an andere Spieler.

McMMO Skills

Level deine Fähigkeiten in Mining, Combat, Woodcutting & mehr.

Tote Chests

Deine Items sind sicher bei Tod – keine verlorenen Inventare mehr.

Mehrere Homes

Setze mehrere Homes und teleportiere dich schnell zwischen deinen Basen.
